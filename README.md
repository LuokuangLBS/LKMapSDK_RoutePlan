## 示例

克隆工程后，运行 `pod install` 

## 系统

iOS 9.0及以上系统

## 安装

LKMapSDK_Base 支持 [CocoaPods](https://cocoapods.org) 安装。
将以下命令添加到你的 Podfile 文件中：

```ruby
pod 'LKMapSDK_RoutePlan'
```
